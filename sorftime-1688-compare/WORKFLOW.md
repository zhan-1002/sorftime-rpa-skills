# 选品工作流设计

## 设计原则

品类数据不可信（混杂品拉偏均值），必须用真实 ASIN 验证。流程分为两个阶段：

### Phase 1：由小见大（样本 → 标准）

```
供应链表格中已验证的产品 ASIN（通过 + 高指标被拒）
       │
       ├─ sorftime checkproduct → 销量/价格/评价/FBA/品类/上架时间
       ├─ 1688 比价 → 采购成本/供应商密度
       ├─ 专利排查 → 风险等级
       └─ 手工标注 → 为什么通过 / 为什么放弃（备注字段）

       ▼
  输出：品类分布画像 + 选品标准阈值
```

### Phase 2：由大及小（品类 → 产品 → 方案）

```
                    ┌──────────────────────────────────────┐
                    │  Step 1. 品类挖掘                      │
                    │  sorftime-market / checkmarket        │
                    │  搜目标大类 → category_filter 过滤     │
                    │  按季节品/全年品分别评分排序              │
                    └──────────────┬───────────────────────┘
                                   │ 得分最高的品类
                                   ▼
                    ┌──────────────────────────────────────┐
                    │  Step 2. 产品发现                      │
                    │  Amazon 搜索品类关键词 → 提取 ASIN     │
                    │  验证 ASIN 确属该品类（标题匹配）        │
                    └──────────────┬───────────────────────┘
                                   │ 验证过的 ASIN 列表
                                   ▼
                    ┌──────────────────────────────────────┐
                    │  Step 3. 竞争分析                      │
                    │  sorftime checkproduct → ASIN 级数据   │
                    │  销量分布/评价分布/价格带/FBA 占比      │
                    │  选出最适合作为模板的竞品 ASIN          │
                    └──────────────┬───────────────────────┘
                                   │ 模板竞品 ASIN
                                   ▼
                    ┌──────────────────────────────────────┐
                    │  Step 4. 竞品深度分析                   │
                    │  Amazon 页面抓取：                      │
                    │  - 价格 / 件数 / 套装组合               │
                    │  - 评论内容（好评/差评关键词）           │
                    │  - 主图 / A+ 内容                      │
                    │  - BSR / 上架时间                      │
                    └──────────────┬───────────────────────┘
                                   │ 竞品完整画像
                                   ▼
                    ┌──────────────────────────────────────┐
                    │  Step 5. 1688 供应链比价                │
                    │  sorftime-1688-compare                │
                    │  - 图搜同款 → 供应商报价               │
                    │  - FBA 利润计算（采购/头程/佣金/FBA费） │
                    │  - 供应商信誉评分                      │
                    └──────────────┬───────────────────────┘
                                   │ 利润数据 + 供应商列表
                                   ▼
                    ┌──────────────────────────────────────┐
                    │  Step 6. 差异化方案                     │
                    │  基于竞品分析 + 差评挖掘：               │
                    │  - 改进方向（件数/材质/包装/功能）       │
                    │  - 重新计算差异化后成本                 │
                    │  - 预估差异化后利润                     │
                    └──────────────┬───────────────────────┘
                                   │
                                   ▼
                    ┌──────────────────────────────────────┐
                    │  Step 7. 输出：可做性判断                │
                    │  - 品类得分                            │
                    │  - 竞品对标                            │
                    │  - 差异化方案                          │
                    │  - 预估利润                            │
                    │  - 供应商链接                          │
                    │  - 专利风险                            │
                    └──────────────────────────────────────┘
```

## 两轨分类

| | 季节性/节日品 | 全年品 |
|---|---|---|
| 典型节点 | 万圣节(9-10月)、圣诞(11-12月)、感恩节 | 全年平销 |
| 备货策略 | 一次性海运，旺季前到仓 | 可补货，关注库存周转 |
| 评分侧重 | 评价壁垒更低、新品占比更高 | FBA 空间、头部集中度 |
| 仓储风险 | 高（过季滞销） | 低 |
